# hermes-windows-sandbox-wrapper

Small Rust wrapper around the Codex Windows sandbox library.

## What It Does

This binary provides a thin CLI layer for:

- executing sandboxed foreground commands
- checking Windows sandbox setup status
- running the setup flow through the upstream Codex setup helper

## Release

This repository publishes a Windows `.exe` through the GitHub release workflow.
